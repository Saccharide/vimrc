# vimrc

This is my personal vimrc file ~

## How to Download
```bash
git clone https://github.com/saccharide/vimrc.git
```                                                                      
## How to Install
```bash
sudo apt install build-essential cmake python3-dev
./INSTALL
cd ~/.vim/bundle/YouCompleteMe
./install.py --clang-complete
```                                                                      
Enjoy :)
